# Project Statement

## Problem Statement
Managing student and course data manually is time-consuming and error-prone. There is a need for a simple system to maintain student records, courses, and enrollments efficiently with easy retrieval and modification.

## Scope of the Project
- Applicable to small colleges or class-level management
- Manage student information (roll, name, email, phone)
- Manage courses (code, title, credits)
- Manage enrollments of students in courses
- Perform search, list, and reporting

## Target Users
- Teachers
- Department staff
- Students (for self-enrollment)

## High-Level Features
1. Student Management (Add/Edit/Delete/List/Search)
2. Course Management (Add/Delete/List)
3. Enrollment Management (Enroll/Unenroll/List)
4. CLI interface with menu-driven navigation
5. SQLite database storage for persistent data
